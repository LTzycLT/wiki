how to tweak Ubuntu 10.10 using Ubuntu Tweak

Ubuntu中的超级兔子——Tweak安装教程



This post will explain about a best application to tweak ubuntu 10.10(Maverick Meerkat) settings without any hassle.The application name is Ubuntu – Tweak ,it’s version 0.5.7 has arrived along with the arrival with Ubuntu 10.10.It’s designed to be very easy to use and intended to make the administration and application management easier.Ubuntu Tweak has good gui design and it’s available free as an open source software.

本文将讲解一个无痛优化ubuntu 10.10(狐獴)设置的最佳程序。这个程序就是Ubuntu-Tweak，随着ubuntu 10.10的发布，它的版本号已经升到了0.5.7。它的设计目地是让系统管理和应用管理更加容易。Ubuntu Tweak 有很好的图形用户界面设计，并且是一个免费自由的开源软件。

How to install Ubuntu Tweak in Ubuntu 10.10

在Ubuntu 10.10中如何安装 Tweak

There are many ways to install Ubuntu Tweak e.g from command line,synaptic package manager or after downloading *.deb package manually from it’s official website.

安装Ubuntu Tweak 有很多方法，比如从命令行安装，用新立得包管理器，或者手动从官网上下载 *.deb 包后安装。

Installing Ubuntu Tweak from Command Line

从命令行安装Ubuntu Tweak

Open Terminal and Type the following command followed by your ubuntu login password.

打开终端，输入下面的命令，并接着输入你的Ubuntu 登录密码。

sudo add-apt-repository ppa:tualatrix/ppa

sudo add-apt-repository ppa:tualatrix/ppa

sudo apt-get update

sudo apt-get update

sudo apt-get install ubuntu-tweak

sudo apt-get install ubuntu-tweak

Download Ubuntu Tweak from it’s website

从网站下载安装Ubuntu Tweak

Other simple way to install ubuntu tweak is to download the ubuntu tweak 0.5.7 from it’s website.

安装Ubuntu Tweak 的另一个简单方法是从网站下载ubuntu tweak 0.5.7

Download Ubuntu Tweak 0.5.7

下载ubuntu tweak 0.5.7

and open it with software center;then click on install button ….that’s all you are done! For stepwise tutorial see here – installing Ubuntu-Tweak

用软件中心打开它；然后点击安装按钮 …… 好了，装好了！更详细的教程看这里--安装Ubuntu-Tweak

Now open Ubuntu Tweak from Applications –> System Tools –> Ubuntu Tweak and configure your ubuntu as you want …If you have any problem …then let us know through comments.

现在可以从这里打开 Ubuntu Tweak，应用->系统工具-> Ubuntu Tweak，然后你可以照你所想去配置你的系统 …… 如果你有什么问题…… 在评论中告诉我们吧。
