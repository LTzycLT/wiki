前 几天看到网上有人安装gnome 我心里就痒痒 ，本来是把我的另一个电脑的mint上安装的 但是那个电脑比较老，配置低，运行时无限卡，没办法就把ubuntu的11.4安装上了gnome3 安装之后看起来他应该是为平板一类的设备开发的 如果是触摸屏勇气来应该是很爽的。
用了两天，总体上还可以就是设置不知倒在哪里，没有gnome2的时候那么简单了，把所有功能都集中在一起了 而gnome3是把常用的功能都显示出来，深层的设置就在应用程序中了。
现在gnome只能从ppa下载 具体方法如下
sudo add-apt-repository ppa:gnome3-team/gnome3
sudo apt-get update
sudo apt-get upgrade
sudo apt-get install gnome-shell
其他的版本ubuntu可以从网上找一下安装的方法。个人建议还是配置高一点的电脑升级吧。要不然升级之后不能用真的很上火。安装完这些还要安装一个软件 来设置gnome的细节，
apt-get install gnome-tweak-tools
就可以设置gnome了 但是我觉得gnome的设置真的很少，不如unity的设置，我现在还不知道怎么改变窗口，边框的样式，还有就是主题，真的很难找啊
大家不在意这些的话可以尝试一下
gnome的功能挺好完 就像按win键是在桌面和 活动 之间切换
还有侧面的dock栏可以放下你喜欢和你常用的软件，和文件夹
下侧的dock栏是你现在的运行的任务窗口
alt+tab是在你的任务之间的切换
当你点击用户名是按alt键就会出现关机
gnome的问题可能就是输入法吧
我之前的系统用的是unity自带的输入法移植没什么问题
安装gnome3后就不知道怎么的在终端和解锁时总不能输入 这很让人头疼 ，可能我安装的比较晚。现在大体上都可以解决了
我 安装其他的输入法也不能用 在输入的时候只能右键然后点击输入法改成无 而且我的系统输入法总是（X输入法）我在网上也没找到解决的办法 ，他们说这是gnome的兼容性问题，可以安装ibus-gtk3 就可以解决 。现在还有就是搜索 应用程序时不能输入中文。只能一个一个程序的找
